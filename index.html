    Gal Gadot Varsano born 30 April 1985, is an Israeli actress and model. At age 18, she was crowned Miss Israel 2004. She then served two years in the Israel Defense Forces as a combat instructor, and began studying law and international relations at IDC Herzliya college while building up her modeling and acting careers. Gadot's first international film role came as Gisele Yashar in Fast & Furious, a role she reprised in subsequent installments of the film franchise. She went on to earn worldwide fame for portraying Wonder Woman in the DC Extended Universe, beginning with Batman v Superman: Dawn of Justice (2016), followed by the solo film Wonder Woman and the ensemble Justice League (both 2017). In 2018, Gadot was included on Time's annual list of the 100 most influential people in the world.

    Questions & Answers

    Question 1
    1. Type of Web
      • Web 1.0- Simply Web

      • Web 2.0- Social Web

      • Web 3.0- Semantic Web
    2. Features
      • Web 1.0- Hyper linking and bookmarking on pages, No communication between server and user, Websites were Static, It allowed only browsing of content.
      • Web 2.0- Better interaction,Introduction of web applications, Everything becomes online and stores on servers, Web 2.0 includes functions like Video streaming, Online documents, etc.
      • Web 3.0- Smart, web based applications and functionalities, An amalgamation of Web technology and Knowledge, Representation (KR).
    3. Definition (According to Burners-Lee)
      • In Web 1.0- Read-only

      • In Web 2.0- Read-write

      • In Web 3.0- Read-write-execute






    Question 2

    Use Case of search query "side effects of medicines"

    • Web 1.0- The user can only read information about side effects of medicines, from the web pages in the browser, which appeared according to the query words.
    • Web 2.0- The user can read about and also comment(write) in forums about side effects of medicines, from his personal experience. Moreover he can see images and videos about it than Web 1.0
    • Web 3.0- In addition to read and write about, the user enter a keyword in google search, and it smartly suggests the related words. It helps in searching the other matched items on the basis of similar attributes. The use of semantics focuses on easy searching than Web 2.0.
